What does this mean?

Every standard Ed25519 public key is a mathematical point that lies exactly on the Ed25519 elliptic curve. This page checks if the provided 32-byte string maps to a valid point on that curve.

If an address is off curve, it is mathematically impossible for a corresponding private key to exist. This provides a unique security property: even if a quantum computer successfully uses Shor's algorithm to derive private keys from public keys, an "off curve" address remains completely uncrackable, simply because there is no private key to find.
